## 🔧 What I build

- **[LAIW](https://github.com/wh1xdy/LAIW)** — a Swedish/EU legal LLM pipeline
  (Mistral 7B + LoRA) over a large legal corpus. Includes a reproducible proof
  of concept: a focused fine-tune fixes the base model's legal-citation
  hallucinations (0/5 → 5/5, verified against riksdagen.se), plus a RAG tool
  layer so the model looks statutes up live instead of guessing.
- **[nuclear_reactor_control_room](https://github.com/wh1xdy/nuclear_reactor_control_room)**
  — a real-time PWR/BWR/RBMK control-room simulator: point-kinetics, thermal
  modelling, xenon poisoning, IAPWS-IF97 steam properties.
- **Lab** — a 42U homelab: 18 servers, a Proxmox cluster and enterprise
  switching, most of it sourced through cold outreach to companies that donated
  the hardware.
